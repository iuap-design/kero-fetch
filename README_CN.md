##  介绍
`kero-fetch`主要提供与服务器交互时所需的方法，需和后台数据模型联合使用



## 安装

首先选择一个目录，克隆项目文件:

```
$ git clone git@github.com:iuap-design/kero-fetch.git
```

然后进入目录安装依赖：

```
$ cd kero-fetch && npm install
```

接下来，执行下面的命令产出：

```
$ npm run product
```
